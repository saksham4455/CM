# Event Rules PDFs

Place your event rule PDFs here with **exactly** these filenames:

| Event            | Filename               |
|------------------|------------------------|
| Treasure Hunt    | `treasure-hunt.pdf`    |
| Innovision 7.0   | `innovision.pdf`       |
| Gaming Arena     | `gaming-arena.pdf`     |
| Hacking Event    | `hacking-event.pdf`    |
| Mind Map         | `mind-map.pdf`         |
| IT Quiz          | `it-quiz.pdf`          |
| Coding Sprint    | `coding-sprint.pdf`    |

These files are served at `/Rules/<filename>` in the browser.
The Rules button on each event card will open the corresponding PDF in a new tab.
